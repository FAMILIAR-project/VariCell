Comparison of IOC, FIFA, and ISO 3166 country codes

{{Refimprove|...}}

This is a comparison of the IOC, FIFA, and ISO 3166-1 three-letter codes, combined into one table for easy reference. Highlighted rows indicate those entries in which the three-letter codes differ from column to column.

This list only includes nations or territories that have been assigned at least one of these three types of country codes. For a more complete list of countries, see list of countries.

The only ambiguous trigraph is "BRN", which stands for Bahrain in IOC code and for Brunei in ISO code.
