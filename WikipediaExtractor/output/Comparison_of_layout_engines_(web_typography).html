Comparison of layout engines (web typography)

The following tables compare support for different font formats and support for CSS3 font resources for a number of layout engines. Web fonts were initially defined (but later dropped) from CSS2, then added into CSS3. The specification is supported by an increasing number of web browsers.<ref>...</ref><ref>...</ref>

Support for CSS font resources

{{Main |...}} CSS3 specified a mechanism for downloading and displaying fonts within a web page.<ref ...>...</ref> This table compare support for that mechanism. Note that Embedded OpenType (EOT)<ref .../> font download does not bring in the entire OpenType font and therefore, the most prominent feature of Open Type, ligature support, does not come with EOT. Trident before version 5.0 supports only EOT. EOT is not sufficient for smartfonts that rely on ligatures.
