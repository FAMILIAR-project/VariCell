Comparison of layout engines (XHTML)

{{Html series}} The following tables compare XHTML compatibility and support for a number of layout engines. Please see the individual products' articles for further information. This article is not all-inclusive or necessarily up-to-date. Unless otherwise specified in footnotes, comparisons are based on the stable versions without any add-ons, extensions or external programs.

Only XHTML 1.1 is considered. XHTML 1.1 is necessarily XHTML 1.0 Strict with more elements and attributes deprecated. XHTML 2.0 was a working draft with no layout engine support, but work on it was abandoned<ref>...</ref> in 2009 in favor of work on HTML5 and XHTML5. XHTML 1.0 and HTML 4 (both served with text/html) are covered in comparison of layout engines (HTML). The comparison of XML capabilities are covered in comparison of layout engines (XML).

Version number, if possible, is provided for feature that is fully supported. Proprietary extensions are not included.
