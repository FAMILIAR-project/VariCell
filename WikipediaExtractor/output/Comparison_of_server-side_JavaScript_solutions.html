Standalone

JavaScript Synchronous, CommonJS, JST (JSP-like template)/Markdown templating, easily extended with C++/Javascript Linux/OSX compatible. "SilkJS enables JavaScript as a scripting language in the server environment. You can think of SilkJS as a command shell, along the lines of Perl or PHP, except the language you write in is JavaScript."<ref>...</ref>
